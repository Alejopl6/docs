# Módulo Inventario

El *Módulo Inventario* es la herramienta central para gestionar, controlar y monitorear las existencias de productos en tu empresa. Permite conocer en tiempo real el stock, ubicación y valor de cada ítem, facilitando operaciones diarias y decisiones estratégicas.

<Note>
Asegúrate de revisar y actualizar el inventario de forma periódica para evitar discrepancias entre el stock físico y el registrado.
</Note>

---

## Ítems de Venta

*Descripción breve:*  
Permite visualizar, crear y editar productos o servicios que están disponibles para la venta.

*Características principales:*  
- Visualización rápida y filtrado de productos.
- Creación de nuevos ítems mediante formularios intuitivos.
- Opciones de edición, desactivación, eliminación y asociación de ítems equivalentes.

<Steps>
  <Step title="Crear Nuevo Producto">
    Accede al formulario para registrar un nuevo producto. Completa la información obligatoria (nombre, referencia, existencias, bodega, precio y categoría) y adjunta una imagen representativa.
  </Step>
  <Step title="Gestión de Acciones">
    Utiliza las opciones de filtrar, importar desde Excel, actualizar masivamente y exportar para mantener el catálogo actualizado.
  </Step>
</Steps>

*Recomendaciones generales:*  
> <Tip>
> Completa la descripción y asigna etiquetas a cada producto para facilitar búsquedas futuras.
> </Tip>

## Grupos de Ítems

*Descripción breve:*  
Facilita la organización de productos agrupándolos según características comunes, lo que permite aplicar descuentos o promociones de manera sencilla.

*Características principales:*  
- Creación y edición de grupos mediante nombre, referencia, impuesto y precio base.
- Asociación de productos a un grupo para una gestión centralizada.

<Card title="Consejo">
  Utiliza grupos para segmentar productos y agilizar campañas promocionales.
</Card>

*Recomendaciones generales:*  
- Revisa periódicamente la estructura de grupos para optimizar la organización del inventario.

## Valor de Inventario

*Descripción breve:*  
Consulta el valor total de tu inventario, observando tanto la cantidad de ítems como su costo inicial.

*Características principales:*  
- Visualización en tiempo real del costo total del inventario.
- Filtros por bodega o estado.
- Exportación de reportes en formato Excel para análisis financiero.

<Info>
Revisa este submódulo mensualmente para monitorear la salud financiera de tu inventario.
</Info>

*Recomendaciones generales:*  
- Realiza reportes periódicos y archívalos para tener un historial financiero detallado.

## Ajustes del Inventario

*Descripción breve:*  
Permite corregir discrepancias entre el stock físico y el registrado, ajustando manualmente las cantidades de productos.

*Características principales:*  
- Selección de la bodega y producto a ajustar.
- Definición del tipo de ajuste: incremento o disminución.
- Registro de cantidad y costo unitario para la auditoría.

<Warning>
Documenta siempre las razones del ajuste para facilitar futuras auditorías y evitar inconsistencias.
</Warning>

*Recomendaciones generales:*  
- Realiza ajustes bajo supervisión y con respaldo de la documentación necesaria.

## Gestión de Ítems

*Descripción breve:*  
Facilita la administración de productos en el día a día, permitiendo operaciones como importaciones masivas y transferencias entre bodegas.

*Características principales:*  
- Importación de nuevos ítems mediante archivo Excel.
- Actualización masiva de productos con un solo archivo.
- Transferencia de productos entre almacenes.

<Steps>
  <Step title="Importar nuevos ítems">
    Descarga la plantilla, completa la información y súbela para agregar múltiples productos de forma simultánea.
  </Step>
  <Step title="Transferencia entre bodegas">
    Selecciona el producto y define la bodega de origen y destino para realizar la transferencia.
  </Step>
</Steps>

*Recomendaciones generales:*  
- Programa actualizaciones periódicas para mantener la información de los productos siempre actualizada.

## Lista de Precios

*Descripción breve:*  
Permite definir y gestionar diferentes listas de precios, ajustadas a criterios como precios mayoristas o minoristas.

*Características principales:*  
- Creación de listas con nombre, asignación de productos y fechas de vigencia.
- Posibilidad de asignar múltiples precios a un mismo producto según la segmentación del mercado.

<Note>
La creación de listas diferenciadas puede mejorar la segmentación y atención a distintos tipos de clientes.
</Note>

*Recomendaciones generales:*  
- Actualiza las listas de precios en función de cambios de mercado y campañas promocionales.

## Bodegas

*Descripción breve:*  
Gestiona todos los almacenes de la empresa, incluyendo su ubicación, tipo y responsable, para una trazabilidad óptima.

*Características principales:*  
- Creación, edición y eliminación de bodegas.
- Definición del tipo de bodega: principal, secundaria o virtual.
- Posibilidad de asignar un responsable para cada bodega.

<Card title="Importante">
  Asigna un responsable a cada bodega para garantizar la correcta administración y seguimiento.
</Card>

*Recomendaciones generales:*  
- Mantén actualizada la información de cada bodega y revisa su capacidad y ubicación regularmente.

## Recomendaciones Generales del Módulo Inventario

- *Revisión periódica:* Verifica el inventario semanalmente para detectar y corregir discrepancias.
- *Copia de seguridad:* Realiza backups antes de ejecutar actualizaciones masivas.
- *Optimización:* Utiliza las listas de precios y la transferencia entre bodegas para mantener un stock equilibrado y actualizado.
- *Auditoría:* Documenta todos los ajustes y cambios para futuras auditorías.
